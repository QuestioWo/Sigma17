Core example programs

This collection of programs uses the core of the Sigma16 architecture. This includes the essential subsystems and instructions.

  • Simple Small and simple programs that illustrate usage of the basic instructions.

  • Arithmetic Demonstrate the arithmetic operations.

  • Arrays Arrays, indexing elements, and traversing arrays.

  • Input/Output Illustrate the simplest ways to read input and write output.

  • Sorting Sorting the elements of an array.

  • Subroutines Basic procedures and functions with simple calling conventions.
